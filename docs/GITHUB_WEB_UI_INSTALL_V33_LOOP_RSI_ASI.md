GoalOS Signoff Pro — v33 Loop → RSI → ASI Superintelligence Console
GitHub Web UI Install Guide

1. Download and unzip the package.
2. Open the unzipped folder.
3. Go to the root of https://github.com/MontrealAI/goalos-signoff-pro.
4. Click Add file → Upload files.
5. Drag the CONTENTS INSIDE the unzipped folder into GitHub.
6. Make sure the hidden .github folder is included.
   - macOS: press Command + Shift + . to show hidden files.
   - Windows: View → Show → Hidden items.
7. Commit to a new branch.

Recommended commit message:

Add GoalOS Loop to RSI to ASI Console Lab v33

Recommended branch name:

add-goalos-signoff-v33-loop-rsi-asi-superintelligence-console

8. Open GitHub Actions.
9. Run:

GoalOS Signoff Pro — Loop to RSI to ASI v33

Recommended settings:

commit_generated_site: true
deploy_pages: true

10. After deployment, open:

https://montrealai.github.io/goalos-signoff-pro/loop-rsi-asi-superintelligence-lab.html

This package is additive. It does not remove existing pages, labs, docs, workflows, or artifacts.
