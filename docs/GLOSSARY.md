# Glossary

- **Acceptance criterion** — a condition that must be satisfied before human acceptance.
- **Evidence Docket** — structured evidence mapped to claims and criteria.
- **ProofBundle** — replayable bundle of evidence, hashes, metadata, and validator output.
- **Mission Receipt** — signed/replayable acceptance record for a bounded mission.
- **Replay** — independent reproduction or inspection path for evidence.
- **Settlement signal** — simulated readiness signal indicating proof passed gates; not value movement on the public site.
- **AGIALPHA boundary** — identity/protocol reference only in public materials unless explicitly documented otherwise.
- **Claim boundary** — the exact limit of what the evidence supports.
