# Website Guide

The public site is a static GitHub Pages-compatible demonstration under `site/`.

## Start routes

- Home: `site/index.html`
- Command center: `site/goalos-v22-v35-command-center.html`
- Public labs: `site/public-demo-labs.html`
- Latest simulator: `site/loop-rsi-asi-superintelligence-mission-simulator-lab.html`
- Public-safe rule: `site/no-user-data.html`

## Visitor journey

Land on home, open the command center, choose a role, inspect proof gates, open Mission 001, explore v22-v35 by chapter, then open v35.

## Public-safe rule

No public page should collect data, connect wallets, call external AI, load analytics, or move value.
