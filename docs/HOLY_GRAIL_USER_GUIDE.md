# Holy Grail Browser Demo — User Guide

## What this is

This is a browser-local demo of the GoalOS proof-gated work loop.

It explains why the important idea is not raw Turing completeness. The important idea is proof-gated open-ended work: a system that can turn autonomous work into evidence, validation, memory, reusable capability, and a harder next mission.

## What to click

Open:

```text
https://montrealai.github.io/goalos-signoff-pro/proof-run-001.html
```

Click:

```text
Launch proof loop
```

The demo advances through:

```text
Mission
Work
Proof
Validation
Verified Experience
Chronicle
Reusable Capability
Settlement Signal
Reinvestment
Harder Mission
```

## What it does not do

It does not ask for user data. It does not upload files. It does not connect a wallet. It does not move funds. It does not run a live settlement. It does not claim realized AGI, ASI, or superintelligence.

## What it is for

It helps users understand the GoalOS thesis in 60 seconds:

```text
Not infinite noise.
Not ungoverned autonomy.
A proof-governed substrate for compounding intelligence.

Now we prove it.
```
