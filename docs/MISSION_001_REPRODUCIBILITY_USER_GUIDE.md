# Mission 001 Reproducibility User Guide

Mission 001 is a public-safe synthetic benchmark reproducibility packet. It is designed to answer the user question: "How do I actually attempt a run?"

## Browser path

1. Open `/mission-001.html`.
2. Review the packet file index.
3. Open `/mission-001-replay.html`.
4. Click `Run Mission 001 replay`.
5. Inspect the replay log and scoreboard.
6. Download the packet JSON.

## GitHub Action path

1. Go to `Actions`.
2. Select `Mission 001 Benchmark Reproducibility Packet`.
3. Click `Run workflow`.
4. Download the artifact named `goalos-mission-001-benchmark-reproducibility-packet`.
