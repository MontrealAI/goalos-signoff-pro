# GoalOS v22-v35 Command Center Guide

Open `site/goalos-v22-v35-command-center.html` first.

The command center explains the whole public-safe suite in one route:

1. choose a role;
2. choose a mission mode;
3. advance deterministic proof gates;
4. inspect a synthetic receipt;
5. open the recommended next lab.

It is button-only and browser-local. It has no forms, no text inputs, no uploads, no wallet connection, no payments, no external AI calls, and zero value moved.

## Recommended paths

- First-time visitor: Command Center → Executive AI Proof Console → v35 Mission Simulator.
- Blockchain/DAO operator: Command Center → v28 Blockchain Credibility → v29 Proof Mandate → v30 Proof Before Settlement.
- Auditor/reviewer: Command Center → v24 Replay → v27 Process Evidence → v35 Receipt.
- Frontier lab/RSI team: Command Center → v32 From Loop to RSI → v34 Control Tower → v35 Simulator.

## What the command center proves

It demonstrates the product logic: objective, mission contract, work, Evidence Docket, replay, validation, human authority, receipt, and settlement-readiness boundary.

## What it does not prove

It does not prove achieved AGI, achieved ASI, production RSI, external certification, legal compliance, financial results, live Mainnet settlement, wallet support, custody, escrow, or deployment authority.
