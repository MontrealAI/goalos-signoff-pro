# v30 Install Guide

## Install with GitHub Web UI

Upload the contents of the unzipped package to the root of the repository.

Make sure these folders appear in the upload:

- `.github/workflows/`
- `config/`
- `docs/`
- `public/research/proof-before-settlement/`
- `scripts/`
- `verification/`

## Then run

`Actions -> GoalOS Signoff Pro — Global Public Demo Labs v22-v30`

Recommended settings:

- `proof_before_settlement_scenario`: `dao-grant-standard`
- `commit_generated_site`: `true`
- `deploy_pages`: `true`

## Expected public route

`https://montrealai.github.io/goalos-signoff-pro/proof-before-settlement-research-lab.html`
