# RSI / ASI-Readiness Boundary

Build the governance institution first, before AGI-scale systems mature.

The public v32-v35 arc is a deterministic governance demonstration: target → emit → filter → atlas → test-plan → eval → insert → promote. OMNI/search may guide allocation and interestingness; proof decides promotion. High novelty increases skepticism. Breakthrough candidates require reproduction, stress tests, persistence gates, dossiers, council review, bounded promotion, or rollback.

ASI-scale systems must not self-authorize. The public demos show readiness gates, not realized ASI or production RSI.
