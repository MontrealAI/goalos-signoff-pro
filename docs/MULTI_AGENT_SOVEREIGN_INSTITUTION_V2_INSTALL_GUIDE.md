This package adds a browser-only public experience for users:

> Large multi-agent systems coordinate to optimal effect when they become proof-governed sovereign institutions.

It enhances the existing repository. It does **not** create a new repository.

## GitHub Web UI install steps

1. Open `MontrealAI/goalos-signoff-pro`.
2. Add or replace the files from this package.
3. Commit to `main` with:

```text
Add Multi-Agent Sovereign Institution demo v2
```

4. Run:

```text
Actions → Website quality gate → Run workflow → main
```

5. Run:

```text
Actions → Multi-Agent Coordination Demo Autopilot → Run workflow → main
```

6. Run:

```text
Actions → Deploy GoalOS Signoff Pro production site → Run workflow → main
```

## What users should experience

Users should be able to click **Launch coordination cycle** and watch a browser-local institutional loop:

```text
Mission → Constellation → Work → Proof → Validation → Chronicle → Capability
```

They should see the trace update, metrics move, agents activate, and the Evidence Docket panel update. No user input is needed.
