# Commercial Validation Plan

## Week 1 target

- 5 pilot users invited.
- 3 Signoffs completed.
- 1 paid pilot conversation.

## Week 2 target

- 10 Signoffs attempted.
- 7 Signoffs completed.
- 3 users say they would use it again.

## What to ask users

1. Was the workflow clear?
2. Did it help you decide whether the work was done?
3. Did the receipt feel useful?
4. What would prevent you from using this on a paid project?
5. Would you pay $49/month for this?

## What to measure

- Time to first project.
- Time from builder submission to decision.
- Number of change-request rounds.
- Number of criteria missing evidence.
- Number of receipts downloaded.
- Whether receipt was attached to invoice or internal approval.
