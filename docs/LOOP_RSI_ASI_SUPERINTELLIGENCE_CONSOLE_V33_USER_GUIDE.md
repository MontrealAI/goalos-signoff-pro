# v33 User Guide — Loop → RSI → ASI Superintelligence Console

## Purpose

This lab makes the GoalOS trajectory easy to understand:

```text
GoalOS proof loop
→ AGI Alpha RSI deterministic invention governance
→ ASI-scale readiness gates
```

It does **not** claim realized ASI. It demonstrates the governance layer needed before any system can responsibly mature toward ASI-scale capability.

## Public-safe interaction model

The console is button-driven only. It uses local JavaScript and deterministic scenario logic. It has no free-text input and makes no external AI call.

## Visitor journey

1. Pick a role.
2. Pick a scenario.
3. Run the proof loop.
4. Escalate to RSI.
5. Inspect ASI gates.
6. Apply policy shocks.
7. Build a dossier.
8. Review/copy the synthetic receipt.

## What the visitor learns

- Proof loops convert work into receipts and memory.
- RSI converts invention into deterministic, replayable state transitions.
- ASI-scale escalation must be governed by gates, councils, stress tests, rollback, and human authority.

## Best one-liner

> No Proof. No Trust. No Settlement. No ungoverned superintelligence.
