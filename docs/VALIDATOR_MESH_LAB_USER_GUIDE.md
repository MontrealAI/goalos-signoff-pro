# Validator Mesh & Falsification Lab — User Guide

Open:

```text
validator-mesh-lab.html
```

Click:

```text
Run verifier mesh
```

The page shows four public-safe synthetic candidates:

```text
C0 — Confident report: rejected
C1 — Proof packet with contradiction: challenged
C2 — Collusive-looking approval: quarantined
C3 — Proof-carrying decision package: accepted for decision review
```

The point is simple:

```text
Proof is not accepted.
Proof is attacked first.
```

GoalOS does not let one evaluator, one score, or one persuasive report become institutional authority. Work must survive source checks, claim support, contradiction review, replay, negative controls, risk boundaries, challenge windows, and human authority.

The demo is 100% browser-local. It does not collect user data, does not ask for files, does not connect a wallet, and does not move value.
