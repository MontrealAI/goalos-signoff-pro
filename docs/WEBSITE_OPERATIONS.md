# Website Operations

The website is generated by GitHub Actions.

## Normal deployment

1. Go to **Actions**.
2. Select **Deploy GoalOS Signoff Pro production site**.
3. Click **Run workflow**.
4. Choose `main`.
5. Wait for green checks.
6. Open `https://montrealai.github.io/goalos-signoff-pro/`.

## Quality workflow

Run **Website quality gate** to build the public site and check user-facing pages before deployment.
