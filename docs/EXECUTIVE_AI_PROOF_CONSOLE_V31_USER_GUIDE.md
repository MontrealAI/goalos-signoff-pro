# Executive AI Proof Console v31 — User Guide

## Purpose

v31 makes the website more obvious, memorable, and useful for first-time visitors.

The existing site already has strong proof labs. v31 adds a guided front door:

1. choose a role;
2. see the trust problem;
3. run a synthetic proof gate;
4. see the required proof package;
5. follow the best next GoalOS lab.

## Why this matters

A blockchain project, DAO, investor, auditor, or enterprise buyer should not need to read every page to understand the core value.

The v31 console gives them the point immediately:

> Blockchain proves the transaction. GoalOS proves the work.

Then it lets them experience the proof loop without entering data.

## Public safety model

The console uses predefined synthetic scenarios and browser-local JavaScript. It has buttons and tabs, but no prompt box, no upload path, no wallet, no analytics, and no payment path.

This preserves the public-site rule while still making the experience interactive and dynamic.
