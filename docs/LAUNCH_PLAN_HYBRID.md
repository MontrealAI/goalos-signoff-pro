# Hybrid launch plan

## Milestone 1 — Commercial private beta

Goal:

```text
10 real Signoffs attempted
7 completed end-to-end
3 users say they would use it again
0 lost evidence
0 broken receipts
0 security incidents
```

Mode: SaaS Basic only.

## Milestone 2 — Verified receipt pilot

Goal:

```text
5 accepted receipts anchored on Sepolia
5 public verification pages pass
0 private-data leakage
```

Mode: Optional Web3 verification.

## Milestone 3 — Paid verified receipts

Goal:

```text
3 customers pay for verified receipts
support burden remains low
receipt verification page is understood by nontechnical users
```

Mode: paid Verified add-on.

## Milestone 4 — Protocol integration

Only after the 48-contract protocol clears independent revalidation, source identity, audit, activation, and user-fund gates.

Mode: AGIALPHA-secured review and later proof-to-payment.
