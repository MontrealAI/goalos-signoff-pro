# Demo Lab User Guide

The Demo Lab is a public-safe, browser-local product tour. It does not ask for user data.

## What to do

1. Open `demo-lab.html`.
2. Click **Load demo mission** on one of the scenario cards.
3. Click **Launch proof cycle**.
4. Watch the six gates advance:
   - Commission
   - Submit
   - Map
   - Review
   - Accept
   - Receipt
5. Open the sample Evidence Docket.
6. Verify the demo Mission Receipt.

## What this demonstrates

The demo shows the product logic:

```text
AI deliverable → mission contract → evidence mapping → human review → accepted state → Mission Receipt
```

It is a browser-local demonstration. It does not process real customer work, accept uploads, connect wallets, or collect analytics.
