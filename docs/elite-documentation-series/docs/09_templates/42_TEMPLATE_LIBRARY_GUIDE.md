> How to use, customize, and govern Signoff templates.

---

## Template governance

Templates are product assets. Treat them as versioned, reviewed content.

## Template fields

- Name
- Best for
- Default deliverables
- Default criteria
- Required evidence
- Suggested reviewer checklist
- Common change requests
- Example receipt language

## Review cadence

Update templates monthly during pilot based on:

- Which criteria users delete
- Which criteria users add
- Which evidence warnings appear most
- Which change requests repeat
- Which templates convert to receipts
