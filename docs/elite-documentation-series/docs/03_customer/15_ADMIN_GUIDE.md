> How workspace owners configure, manage, and monitor Signoff Pro.

---

## Admin responsibilities

- Manage workspace members
- Configure templates
- Set receipt signing policy
- Monitor unresolved projects
- Review audit logs
- Manage billing
- Set verified receipt availability
- Respond to incidents

## Default settings for private beta

| Setting | Recommended value |
| --- | --- |
| Verified receipts | Off by default; Sepolia-only if enabled |
| Public receipt pages | Private unless explicitly shared |
| AI assistant | Enabled with conservative language |
| External reviewers | Invite-only |
| File retention | Pilot default: 180 days unless customer requests deletion |
| Billing | Manual invoice or Stripe test mode until launch gate passes |


## Weekly admin routine

- [ ] Review active projects older than 14 days
- [ ] Review failed receipt verifications
- [ ] Review support tickets
- [ ] Review access exceptions
- [ ] Check backup and storage status
- [ ] Summarize pilot metrics
