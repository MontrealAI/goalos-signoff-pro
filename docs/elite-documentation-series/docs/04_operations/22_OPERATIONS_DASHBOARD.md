## Daily dashboard

| Metric | Why it matters | Healthy signal |
| --- | --- | --- |
| Projects created | Demand signal | Increasing |
| Submissions completed | Builder activation | >70% of invited builders submit |
| Reviews completed | Workflow throughput | <3 days median |
| Receipts issued | Core value delivered | Rising |
| Change requests | Quality feedback | Not zero; not overwhelming |
| Support tickets | Friction signal | Declining per Signoff |
| Receipt failures | Trust risk | 0 |
| Unauthorized access denials | Security signal | Expected low count, investigate spikes |

## Decision thresholds

| Signal | Decision |
| --- | --- |
| 7+ completions and strong repeat intent | Proceed to paid pilot |
| Completions but confusion about setup | Fix onboarding before scaling |
| Users like receipts but dislike workflow | Simplify flow |
| Users ignore receipts | Reposition around approval/change requests |
| Security or evidence-loss incident | Pause pilot and remediate |
