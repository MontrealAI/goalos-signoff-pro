# Repository map

| Path | Purpose |
| --- | --- |
| `README.md` | Public repository front door and proof/claim boundary. |
| `site/` | Generated/static GitHub Pages public demo site. |
| `scripts/` | Builders, generators, verifiers, repository-readiness checks. |
| `config/` | Demo, product, legal, and boundary manifests. |
| `docs/` | Product, operations, legal posture, verification, and historical install guides. |
| `TOOLS/` | Browser-local receipt/key utilities. |
| `contracts/`, `blockchain/`, `services/anchor-relayer/` | Optional verified receipt/anchoring examples; not public wallet flows. |
| `.github/workflows/` | CI, Pages, and demo-generation workflows. |

Historical install guides are retained for audit trail. Prefer `docs/INDEX.md`, `docs/START_HERE.md`, and `docs/CODEX_RUNBOOK.md` for current navigation.
