# Sovereign Experience Stream Lab user guide

Open `sovereign-experience-stream-lab.html` and click **Run experience stream**.

The browser runs a synthetic public-safe demonstration of GoalOS learning authority:

```text
Mission Contract
→ Bounded Action
→ Proof Packet
→ Validator Pass
→ Grounded Reward
→ Quarantine / Reject unsupported events
→ Temporal Option
→ Chronicle
→ Future Routing
```

The important rule is:

```text
Output is not memory.
Activity is not learning.
Only replayable proof-cleared experience can update future routing.
```

You can switch between four public-safe scenarios:

```text
AI research acceptance
Software delivery review
Defensive readiness packet
Policy option proof
```

The lab shows:

```text
accepted events       may become governed experience
quarantined events    remain warnings, not authority
rejected events       cannot become memory
```

The lab also publishes public-safe artifacts:

```text
sovereign-experience-stream-demo-bundle.json
grounded-reward-ledger.json
temporal-option-registry.json
router-policy-update-certificate.json
experience-reanalyze-report.json
```

No user data is requested or processed.
