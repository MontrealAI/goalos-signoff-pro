# Proof Mission Guide

A Proof Mission takes one AI-delivered work package and turns it into an inspectable acceptance record.

## Input

- The work package.
- The decision to support.
- Acceptance criteria.
- Evidence files or links.
- Reviewer / approver.

## Output

- Mission Contract.
- Evidence Docket.
- Claims Matrix.
- Verifier Report.
- Risk Ledger.
- Executive Brief.
- Decision Deck.
- Action Graph.
- Mission Receipt.
- Chronicle Entry.
- Capability Package.

## First pilot

Start with one deliverable and one decision. Do not start by migrating an entire organization.
