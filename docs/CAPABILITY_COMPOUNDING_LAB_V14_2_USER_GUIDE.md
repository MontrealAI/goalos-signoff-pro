# Capability Compounding Lab v14.2 User Guide

The lab demonstrates the core GoalOS compounding idea:

```text
Accepted proof becomes reusable capability.
Reusable capability makes the next mission cheaper, safer, faster, and more verifiable.
```

## How to use it

1. Open `capability-compounding-lab.html`.
2. Choose one public-safe mission class.
3. Click `Run compounding cycle`.
4. Watch three synthetic missions advance.
5. Inspect the Evidence-state trace.
6. Switch between Docket, Capability library, Chronicle, and Scoreboard.
7. Download the demo bundle.

## What the browser shows

- Mission 1 creates the first governed decision.
- Mission 2 reuses the proof pattern.
- Mission 3 handles a harder mission using Chronicle memory.

The metrics show verified work increasing while proof debt, cost index, and risk index decrease.

## What it does not do

The lab does not collect user data, ask for files, connect a wallet, move value, request payment, run analytics, or claim production certification. It is a public-safe synthetic demonstration.
