# User Guide — Proof Gradient Selection Lab

The lab answers one question:

> Which AI work deserves to become institutional memory?

The answer is not “the most persuasive answer” and not “the busiest swarm.” GoalOS requires hard proof gates.

## What happens in the browser

1. Four candidates appear: persuasive output, busy agent trace, partial proof packet, and proof-carrying capability.
2. Click **Run Selection Gate**.
3. Each candidate is checked against mandatory gates.
4. Only the proof-carrying capability passes all gates.
5. The page emits downloadable public-safe demo artifacts.

## Demo artifacts

- `proof-gradient-selection-certificate.json`
- `proof-gradient-evolution-ledger-entry.json`
- `proof-gradient-demo-docket.json`

These are synthetic public-safe artifacts meant to show the shape of GoalOS proof, not to certify a real external run.

## Public-site safety

The page does not ask for any data. It does not contain a form, upload, wallet, login, payment, or analytics script. Everything happens in the browser.
