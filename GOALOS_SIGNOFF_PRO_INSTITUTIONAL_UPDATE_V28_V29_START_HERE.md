## GitHub Web UI install

1. Unzip this package.
2. Open the unzipped folder.
3. In GitHub, open the repository root.
4. Click **Add file → Upload files**.
5. Drag the contents of this folder into GitHub.
6. Include the hidden `.github` folder.
7. Commit to a new branch and open a PR.
8. Run **Actions → GoalOS Signoff Pro — Institutional Website v22-v29**.
