# GoalOS v45 validation expectations

The v45 workflow validates:

- front-and-center command homepage generated;
- command box implemented without blocked HTML form controls;
- chat / Ask GoalOS route window present;
- all pages catalog generated;
- 48-contract atlas route available when config is present;
- exactly one v12 legal rail per HTML page;
- exactly one footer per HTML page;
- production site gate passes;
- public-safe scan passes.
