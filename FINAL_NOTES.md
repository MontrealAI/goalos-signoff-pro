# Final notes

This hybrid package supersedes the pure commercial and pure Web3 packages.

It keeps the commercially useful mainstream workflow and adds optional Web3 verification in a fail-closed way.

Default state:

```text
Basic SaaS: enabled
Signed receipts: enabled
Sepolia anchoring: optional
Mainnet anchoring: disabled
AGIALPHA bonds: disabled
Escrow: disabled
User funds: not authorized
```
