Fast setup checklist

  1. Upload this folder to a new GitHub repo.
  2. Create Supabase project and run SUPABASE_SETUP.sql.
  3. Run npm run setup:keys and add environment secrets.
  4. Deploy the Next.js app.
  5. Create one internal Signoff using the AI Research template.
  6. Complete one receipt from client decision to PDF/JSON export.
  7. Invite 5 pilot users only after the internal receipt verifies.

Keep disabled: Mainnet anchoring, AGIALPHA, escrow, custody, token approvals, and automated acceptance.
