# Code of Conduct

GoalOS Signoff Pro should be built with professional, respectful collaboration.

Expected behavior:

- Be clear and constructive.
- Treat security and privacy concerns seriously.
- Do not publish user data, secrets, or private customer evidence.
- Disagree with ideas without attacking people.
- Escalate safety-sensitive concerns early.

Unacceptable behavior:

- Harassment or abusive comments.
- Sharing credentials or private data.
- Attempting to bypass review gates.
- Making unsupported legal, audit, Mainnet, token, or production-readiness claims.
