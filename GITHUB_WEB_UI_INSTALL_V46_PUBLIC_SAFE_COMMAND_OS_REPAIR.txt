GoalOS Signoff Pro — v46 Public-Safe Command OS Repair
=====================================================

Use this package when the v45 workflow reaches the optional public-safe gate and fails with:

  site/assets/goalos-v40-concierge.js: forbidden form element
  site/assets/goalos-v40-concierge.js: forbidden input element

What happened
-------------
The new v45 command OS is already form-free, but an older v40 concierge asset can still exist in site/assets/ and the repository-wide public-safe scanner checks every HTML and JS file. That legacy asset used HTML form/input markup.

What v46 does
-------------
- Preserves the front-center "Tell GoalOS what you want" command interface.
- Preserves the Ask GoalOS chat/router behavior.
- Replaces the legacy v39/v40 concierge assets with a contenteditable, browser-local, public-safe version.
- Repairs every HTML page to exactly one v12 legal rail and one footer.
- Removes blocked form/input/textarea/select fragments from site HTML and JS.
- Keeps all existing pages navigable and accessible.

GitHub Web UI install
---------------------
1. Download and unzip the ZIP.
2. Open the unzipped folder.
3. Go to the root of MontrealAI/goalos-signoff-pro.
4. Click Add file -> Upload files.
5. Drag the CONTENTS INSIDE the unzipped folder into GitHub.
6. Confirm these are included:
   .github/
   scripts/
   config/
   docs/
   serverless/
   verification/
7. Commit to a new branch or to main.
8. Run:
   Actions -> GoalOS Signoff Pro — Public-Safe Command OS Repair v46
9. Recommended inputs:
   commit_generated_site: true
   deploy_pages: true
   deploy_timeout_ms: 3600000
