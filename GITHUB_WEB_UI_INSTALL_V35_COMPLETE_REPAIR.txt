GoalOS Signoff Pro v35 Complete Repair — GitHub Web UI Install

1. Download and unzip this package.
2. Open the unzipped folder.
3. Select and drag the CONTENTS inside the folder into the repository root on GitHub.
4. Include every folder, especially:
   - .github/
   - scripts/
   - docs/generated-source/
   - site/
   - config/
   - verification/
5. Commit to a new branch.
6. Run:
   Actions → GoalOS Signoff Pro — Complete v22-v35 Repair and Publish
7. Use:
   commit_generated_site: true
   deploy_pages: true
8. After deployment, open:
   https://montrealai.github.io/goalos-signoff-pro/goalos-v22-v35-command-center.html
   https://montrealai.github.io/goalos-signoff-pro/loop-rsi-asi-superintelligence-mission-simulator-lab.html

Common mistake to avoid:
Do not upload only the workflow file. The workflow depends on scripts/ and docs/generated-source/.
