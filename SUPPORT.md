# Support

For the private beta, support should be handled manually and carefully.

## User support categories

- Account access.
- Invitation link problems.
- Evidence upload issues.
- Receipt verification issues.
- Change-request confusion.
- Data deletion or privacy request.
- Security concern.

## Escalate immediately

Escalate to the owner if a user reports:

- private data visible to the wrong person,
- receipt tampering or failed verification,
- leaked secret or key,
- payment or invoice dispute,
- legal or compliance concern,
- security vulnerability.
