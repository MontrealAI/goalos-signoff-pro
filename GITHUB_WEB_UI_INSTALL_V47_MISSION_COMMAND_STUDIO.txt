GoalOS Signoff Pro v47 — GitHub Web UI Install

1. Download the ZIP.
2. Unzip it.
3. Open the folder inside the ZIP.
4. Go to the root of https://github.com/MontrealAI/goalos-signoff-pro
5. Click Add file → Upload files.
6. Drag the contents inside the unzipped folder into GitHub.
7. Make sure .github/ is included.
8. Commit to a new branch.
9. Run Actions → GoalOS Signoff Pro — Mission Command Studio v47.

Recommended settings:
commit_generated_site: true
deploy_pages: true
deploy_timeout_ms: 3600000

Open after deployment:
https://montrealai.github.io/goalos-signoff-pro/goalos-mission-command-studio.html
