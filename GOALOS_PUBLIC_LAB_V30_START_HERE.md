## Best GitHub Web UI install

1. Download and unzip the v30 package.
2. Open the unzipped folder.
3. In GitHub, open the repository root.
4. Click **Add file → Upload files**.
5. Drag the **contents inside** this folder into GitHub.
6. Make sure the hidden `.github` folder is included.
7. Commit to a new branch.
8. Run **Actions → GoalOS Signoff Pro — Global Public Demo Labs v22-v30**.

Recommended workflow options:

`proof_before_settlement_scenario: dao-grant-standard`

`deploy_pages: true`
