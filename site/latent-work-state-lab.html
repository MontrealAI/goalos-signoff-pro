Core law
Search is advisory. Proof decides.
A planner may imagine actions. GoalOS asks which imagined action can become evidence-bearing work under tool scope, budget, validator availability, risk class, replay path, and escalation rule. The accepted result is not a forecast. It is a bounded, inspectable next action.
