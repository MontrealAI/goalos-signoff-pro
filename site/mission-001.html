What is included

The packet is not a concept page. It is a reproducibility kit.

Every artifact has a URL, a purpose, and a deterministic synthetic value so a user can attempt Mission 001 without asking for hidden details.

GitHub path

Run Actions → Mission 001 Benchmark Reproducibility Packet. Download the generated artifact.
