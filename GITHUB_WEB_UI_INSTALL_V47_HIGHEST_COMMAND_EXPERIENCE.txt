GOALOS SIGNOFF PRO v47 — GITHUB WEB UI INSTALL

1. Download and unzip the v47 ZIP package.
2. Open the unzipped folder.
3. Go to the root of https://github.com/MontrealAI/goalos-signoff-pro.
4. Click Add file -> Upload files.
5. Drag the CONTENTS inside the unzipped folder into GitHub.
6. Make sure the hidden .github folder is included.
7. Commit to a new branch.
8. Run: Actions -> GoalOS Signoff Pro — Highest Command Experience v47.
9. Use:
   commit_generated_site: true
   deploy_pages: true
   deploy_timeout_ms: 3600000

After deployment open:
https://montrealai.github.io/goalos-signoff-pro/goalos-universal-command-center.html
