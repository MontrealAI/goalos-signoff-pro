# Start Here — Autonomous Demos

Use these workflows after installing the User Delight Layer.

## For nontechnical users

1. Go to **Actions** in GitHub.
2. Click **User delight demo and preview**.
3. Click **Run workflow**.
4. Choose `main`.
5. Wait for the green checkmark.
6. Download the preview artifacts.

## Generate a public-safe Proof Mission package

1. Go to **Actions**.
2. Click **Generate delight demo Proof Mission**.
3. Click **Run workflow**.
4. Choose `main`.
5. Wait for the green checkmark.
6. Download `goalos-demo-proof-mission`.

The artifact contains:

- mission-contract.json
- claims-matrix.json
- evidence-docket.json
- verifier-report.json
- risk-ledger.json
- decision-state.json
- mission-receipt.json
- public-demo-report.html
- README.md

## Publish to production

1. Go to **Actions**.
2. Click **Deploy GoalOS Signoff Pro production site**.
3. Click **Run workflow**.
4. Choose `main`.
5. Wait for the green checkmark.
6. Open https://montrealai.github.io/goalos-signoff-pro/start.html
