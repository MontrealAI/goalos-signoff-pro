GoalOS Signoff Pro v44 — GitHub Web UI install

1. Download and unzip this package.
2. Open the unzipped folder.
3. In GitHub, open MontrealAI/goalos-signoff-pro at the repository root.
4. Click Add file → Upload files.
5. Drag the contents inside the unzipped folder into GitHub.
6. Make sure .github/, scripts/, config/, docs/, site/ if present, serverless/, and verification/ are included.
7. Commit to a new branch.
8. Run Actions → GoalOS Signoff Pro — Universal Command OS v44.
9. Use commit_generated_site=true, deploy_pages=true, deploy_timeout_ms=3600000.

Important: run the v44 workflow, not the failed v43 workflow first. The v43 workflow is also patched by this package, but v44 is the intended front door.
