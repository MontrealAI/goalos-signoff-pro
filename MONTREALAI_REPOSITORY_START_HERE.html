MontrealAI-owned edition

GoalOS Signoff Pro v1.1

Use this package when the new repository is owned by MontrealAI, for example MontrealAI/goalos-signoff-pro. It is preconfigured with active @MontrealAI CODEOWNERS rules.

Best first move

Create the repository under MontrealAI, upload this package, wait for CI to pass, then protect main.

Click-by-click setup

Create a new GitHub repository owned by MontrealAI.
Use GitHub Desktop or Git to upload every file from this package.
Confirm .github/CODEOWNERS contains active @MontrealAI rules.
Open the Actions tab and wait for CI to pass.
Go to Settings → Rules → Rulesets or branch protection and protect main.
Require pull requests, required checks, and CODEOWNER review.
Only after protections are active, add Supabase, deployment, Stripe, or optional verification secrets.

Never commit these

  • .env, .env.local, or .env.production
  • Supabase service-role keys
  • Stripe live secret keys
  • Wallet private keys, seed phrases, or mnemonics
  • Mainnet deployer keys
  • node_modules/ or .next/
